# **E-commerce**



## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de um sistema de E-commerce, focado na criação de uma interface moderna, responsiva e funcional para exibição e compra de produtos online.

O objetivo principal é aplicar conhecimentos de front-end e evoluir para funcionalidades mais completas, simulando uma loja virtual real.

-  Funcionalidades:
Página inicial com produtos em destaque,
Página de detalhes do produto,
Seleção de tamanho do produto,
Botão para adicionar ao carrinho,
Layout responsivo,
Interface moderna e intuitiva.

## 🛠️ Tecnologias utilizadas:

HTML

CSS

JavaScript


## 🎯 Objetivo

O objetivo deste projeto é desenvolver uma aplicação de E-commerce moderna e funcional, com foco no aprimoramento das habilidades em desenvolvimento front-end e na criação de interfaces intuitivas e responsivas. A proposta é simular o funcionamento de uma loja virtual real, permitindo a visualização de produtos, interação com elementos da página e navegação entre diferentes seções. Além disso, o projeto busca consolidar conhecimentos em HTML, CSS e JavaScript, especialmente na manipulação do DOM e na organização de código. Como evolução, pretende-se integrar funcionalidades mais avançadas, como consumo de APIs, sistema de autenticação de usuários, carrinho de compras dinâmico e integração com um backend, tornando a aplicação cada vez mais próxima de um sistema profissional utilizado no mercado.
